Community Complaint Portal

📌 Project Overview

“Small steps can make a big difference.”
“Clean surroundings lead to a better life.”

Keeping these thoughts in mind, we created this project to help solve local community problems in a simple and user-friendly way.

This project was developed under Track A using HTML and CSS. The main goal of this website is to help people report problems in their area quickly and easily.

---

🚀 Problem Statement

In our daily lives, we often face problems such as:

- Garbage overflow
- Broken streetlights
- Water leakage
- Road damage
- Drainage issues

Many people want to report these problems, but they do not know where or how to complain. Because of this, many issues remain unresolved for a long time.

---

💡 Our Solution

To solve this issue, we created a simple and easy-to-use complaint portal.

The website is designed not only from a developer’s perspective but also from a user’s perspective. We focused on making the platform simple for people who understand both English and Hindi, so anyone can use it comfortably without confusion.

---

📝 Features

- Simple complaint registration form
- Easy-to-understand user interface
- Complaint forwarding through NGO support
- Complaint status tracking system
- Mobile-friendly and beginner-friendly design
- Optional image upload for better issue reporting
📋 Complaint Form Fields
The complaint form contains the following fields:
1. Full Name
2. Phone Number
3. Issue Type
4. Short Description
5. Location
6. Upload Photo (Optional)
After filling in these details, users can easily submit their complaints.
⚙️ Working Process
1. User submits the complaint through the website.
2. The complaint is stored in the system.
3. The complaint is forwarded to the NGO.
4. The NGO communicates with the Municipal Corporation.
5. Users can track the status of their complaint.
This process helps improve transparency and keeps users updated about their issue.
🎯 Project Goal
The main goal of this project is to connect:
- Citizens
- NGOs
- Municipal Corporations
By connecting them together, local problems can be solved faster and communities can become cleaner, safer, and better places to live.
